# t2-code-3
## Summary
An article describing the fresco *School of Athens* by the Renaissance painter Raphael is shown in *Figure 2-56*. Much of the HTML and CSS code has been created for you. Complete the web page design by adding the missing CSS styles.

## Tasks
Save code2-3_txt.html as code2-3.html. Save code2-3_txt.css as code2-3.css. Open the files *code2-3.html* and *code2-3.css* and in the comment section enter your **name** (First + Last) and the **date** (MM/DD/YYYY) into the ```Author:``` and```Date:``` fields of each file.
            
Go to the *code2-3.html* file and within the ```head``` section insert ```link``` elements linking the page to the *code2-3_layout.css* and *code2-3.css* files. Review the contents the files.
            
Open the *code2-3.css* file. For the ```html``` element, set the left padding space to **30** pixels and set the background color to the value **hsla(40, 80%, 75%, 0.5)**.
            
Display all ```h1``` and ```h2``` headings in a **Helvetica**, **Arial**, or **sans-serif** font.
            
For all ```h1``` headings:
            
1. Set the font size to **3em**,
2. Set the line height to **1em**, and
3. Set the bottom margin to **0**.

For all```h2``` headings:
            
1. Set the top margin to **0**,
2. Set the font style to **italic**, and
3. Set the font weight to **normal**.
            
For all inline images, set the top and right margins to **0**, the bottom margin to **10px**, and the left margin to **20px**.
            
For all paragraphs, set the font size to **1.4em** and set the text indent to **1em**. For the first paragraph, using the ```first-of-type``` pseudo-class, set the text indent to **0**. For the first line of the first paragraph, set the font variant to **small-caps**.
            
View the page in your browser to verify that page resembles *Figure 2-56*.
